# DC Atlas Web

Простой HTTP-сервер для просмотра каталога карточек DC Atlas в браузере.

## Описание файлов

| Файл | Назначение |
|------|-----------|
| `catalog_server.py` | HTTP-сервер на встроенном `http.server` |
| `catalog_template.html` | HTML-шаблон страницы (можно редактировать) |

## Быстрый старт

```bash
# На сервере, где установлен бот:
pip install qrcode[pil]  # если не установлен (нужен для импорта конфига)
python3 web/catalog_server.py

# Или указать пути вручную:
python3 web/catalog_server.py --port 9199 --db /var/lib/dc-atlas/dc_atlas.sqlite3
```

## Аргументы

| Аргумент | По умолчанию | Описание |
|----------|-------------|----------|
| `--port` | 9199 | Порт для прослушивания |
| `--bind` | 0.0.0.0 | Адрес для привязки |
| `--db` | авто | Путь к `dc_atlas.sqlite3` |
| `--avatars` | авто | Путь к папке с аватарами |

## Эндпоинты

- `GET /` — HTML-страница с таблицей карточек
- `GET /catalog.json` — JSON-выдача для API
- `GET /avatars/<path>` — статические файлы аватаров
