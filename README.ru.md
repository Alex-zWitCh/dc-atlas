# DC Atlas

**DC Atlas** — это open-source бот для [Delta Chat](https://delta.chat), который работает как каталог сообществ. Он помогает находить группы, каналы, ботов и Telegram-зеркала, а также создаёт read-only зеркала публичных Telegram-каналов прямо внутри мессенджера.

---

## Возможности

### 📋 Каталог сообществ
- Добавление **групп**, **каналов** и **ботов** в единый каталог
- Каждая карточка содержит: название, описание, теги (#хештеги), аватар, ссылку-приглашение
- Поиск по названию, описанию и тегам — `/search <запрос>`
- Просмотр списка свежих карточек (`/new`) и своих карточек (`/my`)
- Открытие карточки по ID — `/open_5`

### 🔄 Telegram-зеркала
- Создание read-only зеркал публичных Telegram-каналов (авто-определение по ссылке `https://t.me/...`)
- Автоматическая публикация новых постов в Delta Chat Channel
- Поддержка **HTTP-прокси** для обхода блокировок Telegram
- Управление зеркалами: приостановка, возобновление, проверка
- Автоматическая очистка старых постов (по возрасту и количеству)

### 🛡️ Модерация и рейтинг
- Репорты на карточки — `/report_<id> <причина>`
- Авто-скрытие при превышении лимита репортов (настраивается)
- Административная панель: скрытие, восстановление, полное удаление

### 🔐 Приватность
- Работает на **chatmail**-сервере — без SMS, без номера телефона
- Сквозное шифрование (Autocrypt) всех сообщений
- Бот не мониторит группы и каналы постоянно
- При добавлении Delta Chat invite бот временно вступает в группу/канал,
  считывает данные для карточки — название, описание, аватар и хештеги —
  после чего выходит
- Telegram-зеркала используют только публичные страницы Telegram

### 💡 Дополнительно
- `/invite` — генерация QR-кода с приглашением бота
- `/help` — интерактивная справка со списком всех команд
- Умный парсинг команд: через пробел **или** подчёркивание (`/open_5` ↔ `/open 5`)
- Rate limiter для защиты от спама
- Поддержка rate limiting для административных действий

### Форматы команд
Команды с ID поддерживают два формата:
- Через подчёркивание: `/open_5`, `/delete_12`, `/report_7 спам`
- Через пробел: `/open 5`, `/delete 12`, `/report 7 спам`

### Авто-добавление по ссылке

Карточки создаются автоматически — просто отправьте ссылку в личный чат с ботом:
- `https://t.me/...` → Telegram-зеркало
- `https://i.delta.chat/#...` → группа, канал или бот Delta Chat

**Важно**: авто-добавление работает только в личных чатах, не в группах/каналах.

## Установка

### Быстрая установка (одной командой)

```bash
curl -sSL https://raw.githubusercontent.com/Alex-zWitCh/dc-atlas/main/setup.sh | sudo bash
```

Скрипт спросит:

1. **Домен chatmail-сервера** (по умолчанию `nine.testrun.org`) — аккаунт бота создаётся автоматически через `/new`
2. **Invite-ссылки администраторов Delta Chat** — вставьте одну или несколько ссылок-приглашений из Delta Chat
3. **HTTP-прокси для Telegram** (опционально)
4. **Ссылку или email саппорта** (опционально)

Email-адреса администраторов извлекаются из invite-ссылок автоматически.

Поддерживаемые ссылки определяются автоматически в личных 1:1 чатах:
- Ссылка на Telegram-канал (`https://t.me/...`) → Telegram-зеркало
- Ссылка-приглашение Delta Chat (`https://i.delta.chat/#...`) → группа, канал или бот

**Важно**: авто-добавление работает только в личных чатах, не в группах/каналах.

Всё остальное скрипт делает сам:
- Клонирует репозиторий в `/opt/dc-atlas`
- Создаёт аккаунт бота на chatmail-сервере
- Устанавливает Python-зависимости
- Инициализирует базу данных
- Создаёт пользователя `dc-atlas`
- Настраивает и запускает systemd-сервис

После установки бот уже работает. Проверить:

```bash
systemctl status dc-atlas
journalctl -u dc-atlas -f
```

### Требования к серверу

- **ОС**: Debian 12 / Ubuntu 24.04+
- **RAM**: от 512 MB
- **Интернет**: исходящий HTTPS (443) и IMAP/SMTP доступ к chatmail-серверу
- **Права**: root (для установки пакетов и systemd)

| Переменная | По умолчанию | Описание |
|-----------|-------------|----------|
| `TELEGRAM_PROXY_ENABLED` | `false` | Включить HTTP-прокси |

## Telegram Mirror: как это работает

Бот может создавать read-only зеркала публичных Telegram-каналов.

1. Пользователь отправляет команду `/add_tg https://t.me/username`
2. Бот парсит публичную HTML-страницу канала Telegram
3. Создаётся Delta Chat Channel и в него публикуются посты
4. Через настроенный интервал (по умолчанию 5 минут) проверяются новые посты

### Прокси для Telegram

Telegram может быть заблокирован в некоторых регионах. Для обхода блокировки:

1. Разверните HTTP-прокси (SOCKS5/HTTP) на внешнем сервере
2. Укажите его в конфигурации:
   ```
   TELEGRAM_PROXY_ENABLED=true
   TELEGRAM_PROXY_URL=http://user:pass@proxy.example.com:1080
   ```
3. **Никогда не указывайте рабочий прокси в публичном коде или README** — адрес прокси задаётся только в локальном `.env`

## Ограничения версии 1.x

- Telegram-зеркала работают только с публичными каналами.
- Приватные Telegram-каналы и invite-only каналы не поддерживаются.
- Парсинг Telegram основан на публичной HTML-странице `t.me/s/<channel>`.
- Если Telegram изменит HTML-разметку, парсер может потребовать обновления.
- Видео, документы и архивы не скачиваются и не пересылаются.
- Фото по умолчанию не пересылаются, если `TELEGRAM_FETCH_MEDIA=false`.
- Delta Chat-группы в каталоге являются обычными группами: все участники могут писать.
- Read-only формат доступен только для Delta Chat Channels.

## Часто задаваемые вопросы

**❓ Нужен ли мне chatmail-сервер?**
Да, боту нужен аккаунт на chatmail-сервере (или любом сервере, поддерживающем Delta Chat). Можно использовать публичные серверы или поднять свой по [документации chatmail](https://chatmail.at/doc/relay/).

**❓ Может ли бот читать личные сообщения?**
Нет. Бот не читает произвольные приватные переписки. Он обрабатывает сообщения, отправленные ему напрямую.
Если пользователь явно отправляет invite-ссылку Delta Chat группы или канала, бот временно вступает в этот чат,
считывает данные для карточки каталога и затем выходит. Постоянного мониторинга группы или канала не происходит.

**❓ Как работает шифрование?**
Все сообщения шифруются сквозным шифрованием (Autocrypt). Ключи генерируются автоматически при первом входе в аккаунт.

**❓ Можно ли запустить несколько ботов?**
Да, для каждого нужен отдельный аккаунт Delta Chat, отдельная директория данных и отдельный процесс.

**❓ Сколько сообщений может обработать бот?**
Бот использует SQLite с оптимизированными запросами. На сервере с 1 CPU и 1GB RAM бот стабильно обслуживает десятки пользователей и сотни карточек.
